- [ ] 1단계: 고급 특성 공학 (Feature Engineering) `<!-- id: 1 -->`
  - [ ] **경기 상태 재구성**: 과거 데이터를 이용해 실시간 점수 차(스코어) 계산 및 변수화.
  - [ ] **엔터티 임베딩**: `player_id`와 `team_id`에 대한 Target Encoding 또는 ID 임베딩 적용.
  - [ ] **맥락(Context) 변수**: 남은 시간, 현재 전/후반 통계 등 추가.
- [ ] 2단계: 딥러닝 접근 (New Track) `<!-- id: 2 -->`
  - [ ] **데이터 파이프라인**: PyTorch/TensorFlow용 시퀀스 데이터셋(Padding 적용) 구축.
  - [ ] **모델 아키텍처**: LSTM 또는 GRU 기반 회귀 모델 구현.
    - 입력: 시퀀스 (x, y, 행동 유형, 선수 ID 등)
    - 출력: 예측 좌표 (x, y)
  - [ ] **학습**: Early Stopping 및 스케줄러 적용하여 학습.
- [ ] 3단계: 모델 고도화 및 스태킹 `<!-- id: 3 -->`
  - [ ] **검증 전략**: `game_id` 기준 GroupKFold 적용 (데이터 누수 방지).
  - [ ] **Optuna 튜닝**: 새로운 변수를 포함하여 트리 모델 재튜닝.
  - [ ] **앙상블**: 트리 모델(XGB/LGBM/Cat)과 딥러닝 모델의 가중 평균 결합.
